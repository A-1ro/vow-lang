# KEI-E1002: undefined type

名前解決: 定義も import もされていない型名を参照した。組み込み型は
`Int` / `String` / `Bool` / `Result<T, E>` / `Option<T>` のみで、それ以外は
同一ファイル内の宣言(`record` / `enum` / `type`)か import が必要。
型エイリアスの循環もこのエラーになる。

```kei
record Order {
  id: CustomerId   // error: undefined type 'CustomerId'
}
```

## 修正

typo なら fix の提案を適用する。外部の型なら `import` する。新しい型なら
`record` / `enum` / `type` で宣言する。
