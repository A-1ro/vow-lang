# KEI-E3005: query observer must be pure

エフェクト検査: `extern query`(純粋観測子)に `uses` を付けた。`query` は副作用のない
**論理的読み取り**であることが定義で、契約式の中から呼べる(spec/kei-spec-v0.2.md §4.3 /
M14 / #45)。エフェクトを伴うなら観測子ではないため `uses` は付けられない。

```kei
import infra.library as Library

extern query Library.availableOf(book: String) -> Int uses Database.Read
// error: query observer 'Library.availableOf' must be pure; a 'query' extern cannot declare 'uses'
```

## 修正

純粋な観測子なら `uses` を外す。実際にエフェクトを伴う外部呼び出しなら `query` を外して
通常の `extern`(`uses` 付き)にする。ただし通常の `extern` は契約式からは呼べない(`KEI-E4004`)。

```kei
extern query Library.availableOf(book: String) -> Int        // 純粋観測子(契約から呼べる)
extern Library.fetchAvailable(book: String) -> Int uses Database.Read   // 作用あり(本体でのみ)
```
