# KEI-E0102: reserved keyword used as identifier

構文解析: 予約語を変数名・パラメータ名・フィールド名などの識別子として使った。

予約語: `module import as type record enum func uses requires ensures let if
else fail return tagged true false implies`

```kei
let type = 1   // error: 'type' is a reserved keyword
```

なお `.` の後ろのメンバー名は文脈キーワードとして予約語と同綴りを許す
(例: `Audit.Log.record(...)`)。

## 修正

別名にする(fix は末尾 `_` の付与を提案する)。
