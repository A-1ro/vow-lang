# KEI-E3003: duplicate extern signature

エフェクト検査: 同じ外部パス(`Database.fetchBalance` など)に対する `extern`
署名が複数宣言されている。外部境界の署名は 1 パス 1 つに保つ(複数あると、
どの戻り型・エフェクトが正なのか曖昧になる)。

```kei
import infra.time as Time

extern Time.now() -> Int uses Clock
extern Time.now() -> Int uses Clock   // error: duplicate extern signature for 'Time.now'
```

## 修正

重複した `extern` 宣言を削除する。署名を変えたい場合は 1 つだけ残して編集する。
外部境界の署名は spec/kei-spec-v0.2.md §2 を参照。
