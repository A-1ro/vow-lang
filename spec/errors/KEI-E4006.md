# KEI-E4006: invalid seed file (inputs only)

契約ベース PBT のシードファイル(`<stem>.seeds`)の文法エラー。シードは **入力だけ**を供給する
(spec/kei-spec-v0.2.md §5 / M15 / #26)。文法は `seeds for <fn> { input { <field>: <literal> } }`
のみで、**期待値を書く構文は存在しない**——`expected` / `output` / `result` を書くと弾かれる。
これは捏造不能性(§5.1: オラクルは契約だけ)を**言語構造で**保証するための制約。

```text
seeds for decrementAvailable {
  expected { result: 0 }
  # error: 'expected' is not allowed: seed files supply inputs only —
  #        the oracle is the contract (ensures), never the seed
}
```

## 修正

シードには `input { ... }` で**入力だけ**を書く。期待値はシードに書かず、`ensures`(契約)に書く。
テストが思うように書けないときは、シードを増やすのではなく**契約を強化**する。

```text
seeds for decrementAvailable {
  input { available: 1 }
  input { available: 42 }
}
```
