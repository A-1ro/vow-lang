# KEI-E2003: unknown or misused enum variant

型検査: enum に存在しないバリアントを参照したか、バリアントの構築形が
ペイロードの種類と合っていない。Unit バリアントは `E.V`、位置ペイロードは
`E.V(...)`、名前付きフィールドは `E.V { ... }` で構築する。enum 名だけでの
構築(`E { ... }` / `E(...)`)はできない。

```kei
enum Shape {
  Dot
  Line { length: Int }
}

func demo() -> Shape {
  return Shape.Dott   // error: no variant 'Dott' on enum 'Shape'
}
```

## 修正

typo なら fix の提案を適用する。ペイロードの種類に応じた構築形に書き換える。
