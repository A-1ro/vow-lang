# KEI-E4002: contract-only construct misused

契約検査: `ensures` 専用の構文を別の場所で使った。`old(expr)`(呼び出し前の
値の参照)と `result`(戻り値の参照)は事後条件にしか意味がないため、
`requires` や関数本体では使えない(spec §2.1)。

```kei
func deposit(balance: Int, amount: Int) -> Int
  requires old(balance) >= 0   // error: 'old(...)' is only available in 'ensures' clauses
{
  return balance + amount
}
```

## 修正

その条件を `ensures` に移すか、`requires` ではパラメータの現在値を直接参照する。
