# KEI-E2001: type mismatch

型検査: 式の型が文脈の期待型と一致しない。`let` 注釈との不一致、`if` 条件が
`Bool` でない、`return` と戻り値型の不一致、引数の型・個数違い、
`Result` と `Option` の混同、呼び出せない値の呼び出しなどを含む。
暗黙の型変換は存在しない(spec §1 曖昧さゼロ)。

```kei
func loadUser(known: Bool) -> Result<Int, String> {
  return 3   // error: expected 'Result<Int, String>', found 'Int'
}
```

## 修正

fix の提案に従う。`Result` / `Option` の中身を裸で返している場合は
`Ok(...)` / `Some(...)` で包む fix が機械適用できる。
