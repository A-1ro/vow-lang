# KEI-E3001: effect used but not declared

エフェクト検査: `uses` 節に宣言していないエフェクトを行使した。エフェクトは
ケーパビリティであり、呼び出し先の宣言エフェクトは呼び出し元へ推移的に
伝播する(spec §3.1)。宣言は階層で包含してもよい(`uses IO` は全 IO、
`uses Database` は `Database.Read` / `Database.Write` を包含する — spec §3.2)。

```kei
func writeRow(id: Int) -> Bool
  uses Database.Write
{
  return true
}

func save(id: Int) -> Bool {
  return writeRow(id)   // error: effect 'Database.Write' used but not declared in 'uses' clause of 'save'
}
```

## 修正

fix の提案に従い、呼び出し元の `uses` 節に不足エフェクトを追加する
(既存の `uses` 節があれば `, Effect` の追記が機械適用できる)。
