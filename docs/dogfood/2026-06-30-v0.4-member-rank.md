## お題(プロンプト本文 — Kei 語彙ゼロ)

> ある利用者の月次トランザクションリスト(各トランザクションは整数の金額で、マイナスは返金を表します)を受け取り、その人の会員ランクを判定する関数を 1 つ書いてください。
>
> 1. 金額が 0 未満(返金)のトランザクションは集計対象から **除外** する。
> 2. 残ったトランザクションのうち、「合計金額が 100000 以上」または「件数が 30 件以上」のいずれかを満たすなら「Gold」を返す。
> 3. そうでなければ、「合計金額が 30000 以上」または「件数が 10 件以上」のいずれかを満たすなら「Silver」を返す。
> 4. それ以外は「Bronze」を返す。
> 5. 集計対象が 1 件も残らない場合は失敗として扱い、エラーを返す。

## 結果

**成功**(初回 `kei_check` PASS)。サブエージェント自己採点: 9/10。

### サブエージェントが書いた .kei

```kei
module member.rank

enum RankError {
  NoTransactions
}

func isPositive(amount: Int) -> Bool {
  return amount >= 0
}

func sumAmounts(acc: Int, amount: Int) -> Int {
  return acc + amount
}

func memberRank(transactions: List<Int>) -> Result<String, RankError> {
  let positive = transactions.filter(isPositive)
  if positive.isEmpty() {
    return Err(RankError.NoTransactions)
  }
  let total = positive.fold(0, sumAmounts)
  let count = positive.length
  if total >= 100000 || count >= 30 {
    return Ok("Gold")
  }
  if total >= 30000 || count >= 10 {
    return Ok("Silver")
  }
  return Ok("Bronze")
}
```

## 観測された摩擦

1. **`length` がプロパティか メソッド呼び出しか曖昧** — SKILL.md コンビネータ列挙の `length`・`isEmpty()` で「括弧の有無で推論」できたが明示コード例が無い。
2. **`List<Int>` のようなプリミティブ型リストの例が `examples/` に無い** — 既存例は record 要素のみ。「`(Int) -> Bool` / `(Int, Int) -> Int` の named function でも `filter`/`fold` が受け取れるか」を推測で書いた(結果通った)。
3. **取説 / 検証ツールへの到達は迷いなし** — SKILL.md パス提示と「最初にこれを読め」プロンプトのおかげ。

## 助かった点(原文要約)

- **「演算子はこれで全部」セクションに `||` が明記、`&&` 無しも明示** — OR 条件を迷わず書け、`&&` でハマらず。
- **`filter`/`fold` の引数は「名前付き純粋関数の参照」と明示**、lambda は v0.4 以降と記載 → 素直に named function に流れた。
- **`Result` / `Ok` / `Err` の具体例が SKILL.md に揃っている**。
- **MCP `kei_check` が source テキスト直接受け付け** → ファイル書き出し前にチェックできる。

### 重要な観察: lambda(M25)が 2 回連続で named function に流れた

Round 1 / Round 2 ともに、サブエージェントは SKILL.md を読んだ後 `filter`/`fold` の引数に named function を選んだ。Round 2 のフィードバックには「ラムダ構文が v0.4 以降と書かれていたので、素直にヘルパー関数を切り出す方針がすぐ決まった」と明記。

つまり SKILL.md に lambda の存在は書いてあるが、**「いつ lambda を使うべきか」の誘導が無い**(または named function を推す書き方になっている)。これは取説の「lambda 存在告知」と「実用例提示」のバランス問題。
